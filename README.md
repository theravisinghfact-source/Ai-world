# 🌐 AI World - Duniya ki AI Jankari

Swagat hai **AI World** mein! Yeh ek aisa platform hai jahan duniya bhar ki Artificial Intelligence (AI) se judi nayi updates aur jankari ek hi jagah par milti hai.

## 🚀 Features
* **Real-time Updates:** Firebase Database ka use karke naya data turant dikhta hai.
* **Responsive Design:** Mobile aur Computer dono par sahi se chalta hai.
* **Tech Focused:** Sirf AI aur Future Tech se judi jankari ke liye banaya gaya hai.

## 🛠️ Technologies Used
* **Frontend:** HTML5, CSS3 (Modern UI)
* **Backend/Database:** Firebase Firestore
* **Hosting:** GitHub Pages
